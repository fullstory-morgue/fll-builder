sidux 2007-03

In Hinblick auf die finale Veröffentlichung von sidux 2007-03 Γαῖα und trotz der durch die Urlaubssaison bedingten Ausfälle, konnten einige weitreichende Transitionen zur Vermeidung harter Abhängigkeiten von gtk+ 1.2 vollendet werden, welches nicht nur einen signifikanten Anteil Speicherplatz auf dem ISO einfordert, sondern auch zusätzlichen Aufwand für Wartung und Sicherheitssupport bedeutete.

Nun sind wir stolz die dritte Preview für sidux 2007-03 Γαῖα zu veröffentlichen und erwarten das vollausgestattete finale Release in etwa einer Woche.

Release Notes für Γαῖα

Nachdem sich Kernel und die Neue Livebootinfrastruktur als stabil bewährt haben, ist Preview 3 nun auf Regressionstests der teilweise neu geschriebenen und massiv überholten Konfigurationstools und die Verbesserung von Übersetzungen und Lokalisierungen ausgelegt, Wie üblich erscheint dieses Preview Release nur als KDE-Lite Variante, mit einer kleinen (≈440 MB), aber voll funktionalen KDE basierten Paketliste, um das Testen und eventuelle xdelta Uploads zu erleichtern. sidux ist eine vollausgestattete Debian sid basierende Live CD mit einem speziellen Fokus auf Festplatteninstallation, einem sauberen Upgrade Pfad in sid und zusätzlichem Hard- und Software Support.

Das vollausgestattete final Release von sidux 2007-03 "Γαῖα" wird in etwa einer Woche fertig sein zwischenzeitlich gibt es X-Deltas nach Bedarf.

Jetzt zu den interessanten Themen, was steht nun auf dem Plan:

  • amd64 (AMD64, Intel Core2, neuere 64 bit fähige AMD Sempron and Intel Pentium 4 CPUs (achtet auf das «lm»-Flag in /proc/cpuinfo oder nutzt infobash -v3) und i686 (Pentium pro, Pentium II oder neuer).
  • Debian sid.
  • kernel 2.6.22.2 (smp, hard preemption).
  • minimales, aber voll funktionelles KDE 3.5.7 (en + de).
  • vollkommen überholte init Sequenz.
  • komplett überholtes Installer Backend.
  • neues Artwork gestaltet von cleary und dem sidux art team.
  • offline manual for en + de direkt auf der CD, online Handbücher für weitere Sprachen auf: http://manual.sidux.com/ und verfügbar via apt; ein grosses Dankeschön geht wieder einmal an das DoKumentations- und Übersetzerteam!
    Bitte beachtet, dass das Offline Handbuch nur durch die Live-CD oder im installierten System zu erreichen ist.
  • viele Änderungen im und am Handbuch.
  • iwlwifi3945/ iwlwifi4965 1.0.0-1.
  • rt2x00 2.0.6+git20070810 Support für RaLink Karten die wpasupplicant unterstützen, bitte entfernt /etc/modprobe.d/mac80211 um es dem RaLink legacy Treiber vorzuziehen
  • Support für RealTek RTL8187 wlan Karten.
  • libata Support für parallel ATA Systeme.
  • native vmlinuz-$KVERS und initrd.img-$KVERS Namensschemata werden jetzt auf der Live CD benutzt, bitte schaut ins Handbuch wegen der fromiso Syntax Änderungen.
  • erweiterte Hardware Support Libraries.
  • Support für "smxi", welches "sm" und "du-fixes-h2.sh" ersetzt.
  • Bugfix für die Tastaturlayouteinstellungen unter X.
  • die USB Scanner Unterstützung wurde in Kernel ≥2.6.22.1-slh-smp-7/ ≥2.6.22.1-slh64-smp-7 korrigiert.
  • Korrektur des XDG Menüeintrages für My-PPPoE Conf.
  • Installation von siduxcc und kooka.
  • Dank konzertierten Teamanstrengungen, konnten die Abhängigkeiten von Xdialog und damit gtk+ 1.2 aufgelöst werden.
  • Unterstützung für Agere Systems et131x Ethernetkarten.
  • fw-detect für Hardware, die non-free Pakete braucht.
  • rt73-legacy Treiber.
  • memtest86+
  • iputils-ping.

Was hat es nicht in diese Preview geschafft?:

  • Wir arbeiten mit einer kleinen Packagelist im Moment, um die Tests zu erleichtern, ebenso die xdeltas, und um etwas Raum für Diskussionen zu lassen, welche eurer Paketwünsche bei der finalen Version berücksichtigt werden können.
  • neue X.org Live Konfiguration.
  • Ergänzungen im Handbuch für Γαῖα.

Hinweise für Hardware, die non-free Pakete braucht:

sidux enthält nur dfsg freie Software, deshalb ist es sinnvoll, contrib/ non-free in die /etc/apt/sources.list aufzunehmen und Internetzugang sicherzustellen. 

# See sources.list(5) for more information, especially
# Remember that you can only use http, ftp or file URIs
# CDROMs are managed through the apt-cdrom tool.

# Unstable
deb http://ftp.de.debian.org/debian unstable main contrib non-free
# deb-src http://ftp.de.debian.org/debian unstable main contrib non-free

# sidux
deb http://sidux.com/debian/ sid main contrib non-free firmware fix.main fix.contrib fix.non-free
#deb-src http://sidux.com/debian/ sid main contrib non-free firmware fix.main fix.contrib fix.non-free

Die benötigte Firmware kann entweder der unten folgenden Auflistung entnommen, oder mit dem neu entwickelten fw-detect script (paketiert in configure-networkcard) werden.

  • ATi Radeon Grafik:
    3d Beschleunigung für ältere Karten bis r35x sollte funktionieren, neuere Radeon X1xxx Karten brauchen non-free Treiber für beschleunigte Performance, die bisher noch nicht kompatibel mit X.org 7.2 sind. Abhängig vom jeweiligen Chipsatz "xmodule=vesa" oder "xmodule=fbdev" als Cheatcode könnte helfen bei Radeon X1xxx Modellen.
  • Atheros/ "madwifi" wlan:
    m-a a-i madwifi
  • Atmel AT76c50x 11 MBit/s wlan:
    apt-get install atmel-firmware
  • Broadcom/ bcm43xx wlan:
    apt-get install bcm43xx-fwcutter
  • Eagle USB ADSL modem:
    hole die Firmware von http://eagle-usb.org/ueagle-atm/non-free/ und kopiere sie nach /lib/firmware/.
  • DVB Firmware für verschiedene full featured DVB TV Karten (die meisten budget Karten brauchen das nicht):
    hole die benötigte Firmware (nachschauen mit dmesg) von: http://www.linuxtv.org/downloads/firmware/ und kopiere sie nach /lib/firmware/.
  • Intel ipw2100, 11 MBit/s wlan:
    apt-get install ipw2100-firmware
  • Intel ipw2200, 54 MBit/s wlan:
    apt-get install ipw2200-firmware
  • Intel ipw3945/ iwlwifi, 54 Mbit/s wlan:
    apt-get install firmware-iwlwifi
    bitte entferne iwlwifi-ucode und iwlwifi-3945, falls bereits installiert.
  • Intel ipw4965/ iwlwifi, draft-n wlan:
    apt-get install firmware-iwlwifi
    bitte entferne iwlwifi-4965, falls bereits installiert.

    feedback zu iwlwifi 3945 und iwlwifi 4965 ist immer willkommen.
  • Intersil prism54, 54 MBit/s wlan:
    hole die Firmware von http://prism54.org/firmware/ und kopiere sie nach /lib/firmware/.
  • nVidia graphics:
    3d Beschleunigung geht mit freien Treibern noch nicht, bitte benutzt get-sidux-binary-gfx um Install Skripte für diese Karten zu bekommen.
  • RaLink rt61 54 MBit/s wlan:
    apt-get install rt61-fwcutter
  • RaLink rt73 54 MBit/s wlan:
    apt-get install rt73-fwcutter
  • Texas Instruments ACX100 (22 Mbit/s)/ ACX111 (54 MBit/s) wlan:
    hole die firmware von http://www.kazer.org/acx-firmware-20060207.tar.bz2 und kopiere sie nach /lib/firmware/.
  • ZyDAS zd1201 11 MBit/s wlan:
    hole die firmware von http://surfnet.dl.sourceforge.net/sourceforge/linux-lc100020/zd1201-0.14-fw.tar.gz und kopiere sie nach /lib/firmware/.
  • ZyDAS zd1211 54 MBit/s wlan:
    apt-get install zd1211-firmware

Wir versuchen, Pakete für legal vertreibbare Firmware in unseren non-free Repositories zur Verfügung zu stellen, aber nicht alle Anbieter erlauben dies.

Wie kann man mitwirken?

sich bei sidux zu engagieren ist einfach, besucht uns im Forum or even better on IRC (irc.oftc.net, port 6667, channel #sidux) und diskutiert Eure Patches/ Vorschläge oder Erweiterungen. Wir diskutieren gerne jegliche dfsg- free Patches und Beiträge. Besonders im bereich alternativer Window Manager (gnome, xfce), mobiles Netzwerken (GPRS/ UMTS, Bluetooth, aber auch ISDN, PPP, PPPoE) oder generelle Tasks wie Übersetzungen/Lokalisationen und Ähnliches.
