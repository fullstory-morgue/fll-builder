Willkommen zum sidux Gaia Release - Bitte lesen Sie diese Versionshinweise

Seien Sie bereit!

sidux basiert auf Debian 'sid' GNU/Linux, und ist den Grundwerten und dem Gesellschaftsvertrag von Debian verpflichtet.

Um die neuesten Programmversionen inklusive Linux-Kernel zu erhalten, muss nicht mehr länger auf einen neuen Release gewartet werden. Nach der Erstinstallation muss nur mehr ein sogenannter dist-upgrade, d. h. ein systemweiter Update der Software von Debian und sidux durchgeführt werden.

Dies bedeutet, dass eine Neuinstallation eines aktuelleren Releases alle sechs bis zwölf Monate hinfällig ist, wenn sidux installiert ist. Ein wöchentlicher oder monatlicher dist-upgrade aktualisiert die Installation mit Ausnahme des Kernels, welcher bei Bedarf separat auf eine neue Version gebracht werden kann.

Wie dem auch sei: Eine Installation von sidux dauert nicht länger als 20 Minuten.

Auf 'sidux-Art' wird das System immer aktuell sein und alles bieten, was zum gegebenen Zeitpunkt sidux und Debian gemeinsam im Angebot haben. Auch ohne Neuinstallation.

Herzlich willkommen bei sidux!

Versionshinweise für Gaia

Nach drei Monaten Entwicklung geben wir mit Freuden die sofortige Verfügung von sidux-2007-03 "Γαῖα" (Gaia) bekannt, das für amd64- und i686-Systeme ausgeliefert wird – in einer ≈ 425 MB großen lite-KDE- und einer ≈ 690 MB großen full-KDE-Variante. Sidux ist eine vollständig Debian-sid-basierte Live-CD mit einem speziellen Fokus auf Festplatteninstallationen, einen sauberen Upgrade-Pfad in sid und zusätzlichem Hardware- und Softwaresupport.

Unser drittes offizielles sidux-Release konzentriert sich auf die Überholung der früheren Bootsequenz und auf die Anpassung an eine größere Auswahl an Desktop Environments und Fenstermanager. Weitere aktuelle Anstrengungen betreffen den Bereich der zweiten Bootstufe, Lokalisationen, toolkit abstractions/harmonisations und den Installer. Da dieses Release nur in zwei KDE-Varianten ausgeteilt wird, suchen wir interessierte Maintainer, die für andere Desktop-Environments oder Fenstermamager tweaking-Support und spezielle Releases beisteuern. Das ISO ist komplett debian-sid-basiert, angereichert und stabilisiert mit sidux' eigenen Paketen und Skripten. Es kommt mit Kernel 2.6.22.x, der auf dem neuesten Vanilla-Kernel basiert, zusammen mit verschiedenen Patches für verbesserten Hardware-Support, speziell für Wireless-LAN-Geräte (mac80211).

Nun zu den interessanten Dingen, z.B. was heute auf der Karte steht:

  • amd64 (AMD64, Intel Core2, neuer 64 bit fähige Pentium 4 CPUs (achten Sie auf das "lm"-flag in /proc/cpuinfo oder benutzen Sie infobash -v3) und i686 (pentium pro/ pentium II oder neuer).
  • debian sid
  • kernel 2.6.22 (smp, hard preemption, mac80211).
  • KDE 3.5.7 (en + de).
  • komplett überarbeitete frühere Bootsequenz.
  • weithin modifizierter Installer und grafisches frontend.
  • verbesserter Installer für meta Pakete.
  • neues Artwork, erstellt von cathbard und dem sidux art team.
  • offline Handbuch für en + de direkt von der CD, online Handbücher für mehr Sprachen unter http://manual.sidux.com/ und verfügbar via apt; ein großer Dank geht an die documentation and translation teams!
  • zahlreiche Fixes and Verbesserungen für den Installer und den meta-Installer.
  • Installer Fixes für fromiso boots.
  • Veränderungen bezüglich des access concentrator probing für PPPoE links.
  • fixed xawtv Konfigurations-frontend.
  • kleinere Verbesserungen im Artwork.
  • viele Veränderungen im Handbuch
  • iwlwifi 0.25
  • ndiswrapper Updates.
  • Unterstützung für ATi SB700 Chipsets.
  • MPC52xx Unterstützung.
  • Leistungserhöhungen für Intel ICH7 based IDE controller.
  • verbesserter Support für SCSI-Geräte.

Tipps zu Hardware, die unfreie Treiber benötigt:

Auf Grund der schwierigen Rechtslage liefert sidux auf der Live-CD nur DFSG-freie Software aus.

Daher könnte es vom Anwender gewünscht sein, contrib/non-free zur /etc/apt/sources.list hinzuzufügen:

# See sources.list(5) for more information, especialy
# Remember that you can only use http, ftp or file URIs
# CDROMs are managed through the apt-cdrom tool.

# Unstable
deb http://ftp.de.debian.org/debian unstable main contrib non-free
# deb-src http://ftp.de.debian.org/debian unstable main contrib non-free

# sidux
deb http://sidux.com/debian/ sid main contrib non-free firmware fix.main fix.contrib fix.non-free
#deb-src http://sidux.com/debian/ sid main contrib non-free firmware fix.main fix.contrib fix.non-free
  • ATI Radeon: 3D-Beschleunigung für ältere Karten bis r35x sollte funktionieren. Neuere Grafikkarten (Radeon X1xxx) benötigen unfreie Treiber für eine beschleunigte Leistung, die jedoch noch nicht mit X.org 7.2 kompatibel sind.
  • Atheros/ "madwifi" wlan: In der Konsole: 
    m-a a-i madwifi
  • Atmel AT76c50x 11 MBit/s wlan: In der Konsole: 
    apt-get install atmel-firmware
  • Broadcom/bcm43xx wlan: In der Konsole: 
    apt-get install bcm43xx-fwcutter
  • Eagle USB ADSL modem: Die Firmware gibt es bei http://eagle-usb.org/ueagle-atm/non-free/ und wird gespeichert unter: /lib/firmware/

  • DVB Firmware für diverse DVB-TV-Karten (die meisten einfacheren Karten benötigen das nicht). Die benötigte Firmware (dmesg ansehen) bei http://www.linuxtv.org/downloads/firmware/ herunterladen und unter /lib/firmware/ speichern.

  • Intel ipw2100, 11 MBit/s wlan: 
    apt-get install ipw2100-firmware
  • Intel ipw2200, 54 MBit/s wlan: 
    apt-get install ipw2200-firmware
  • Intel ipw3945/ iwlwifi, 54 Mbit/s wlan: 
    apt-get install iwlwifi-ucode-3945
  • Intersil prism54, 54-MBit/s-WLAN: Firmware herunterladen von http://prism54.org/firmware/ und unter /lib/firmware/ speichern.

  • nVidia Grafikkarten: 3D-Beschleunigung ist bisher mit den freien Treibern nicht möglich. Unter Verwendung von get-sidux-binary-gfx lassen sich die unfreien Treiber jedoch nachträglich installieren.

  • RaLink rt61 54-MBit/s-WLAN: 
    apt-get install rt61-fwcutter
  • Texas Instruments ACX100 (22 Mbit/s)/ ACX111 (54 MBit/s) WLAN: Firmware bei http://www.kazer.org/acx-firmware-20060207.tar.bz2 herunterladen und unter /lib/firmware/ speichern.

  • ZyDAS zd1201 11 MBit/s wlan: 
    apt-get install zd1201-firmware
  • ZyDAS zd1211 54 MBit/s wlan: 
    apt-get install zd1211-firmware

Wir untersuchen noch die Möglichkeit, Treiber mancher Geräte mitzuliefern. Der rechtliche Status dieser Treiber ist leider häufig kompliziert.

Wie man mithelfen kann

An sidux mitzuwirken ist einfach; melde dich im Forum an, oder noch besser: Betritt unseren IRC-Channel (irc.oftc.net, port 6667, channel #sidux) und diskutiere deine Patches, Vorschläge oder Ergänzungen. Wir werden gerne über alle dfsg-freien Patches und Beiträge beraten.
